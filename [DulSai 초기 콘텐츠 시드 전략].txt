[DulSai 초기 콘텐츠 시드 전략]

목적:
- 첫 방문자가 빈 앱처럼 느끼지 않게 한다.
- DulSai의 말투와 분위기를 보여준다.
- 사용자가 어떤 글을 써야 하는지 자연스럽게 학습하게 한다.

구성:
- 남성 샘플 프로필 10명
- 여성 샘플 프로필 10명
- 10대, 20대, 30대, 40대, 50대, 60대 이상 분산
- 각 사용자별 글 2~3개
- 일부 글에는 답글/대화 흐름 포함

주의:
- 실제 사용자인 척하면 안 된다.
- 내부적으로는 반드시 sample_user / seed_profile 로 구분한다.
- 실제 매칭/대화 대상이 되면 안 된다.
- 사용자에게는 “예시 글”, “DulSai 샘플”처럼 자연스럽게 표시하는 게 안전하다.

샘플 콘텐츠 방향:
- 하루의 생각
- 외로운 순간
- 누군가에게 하지 못한 말
- 요즘 자주 드는 마음
- 사랑에서 중요하게 생각하는 것
- 내가 편안함을 느끼는 사람

DB 설계:
- users.is_sample = true
- posts.is_sample = true
- comments.is_sample = true
- conversations.is_sample = true
