[DulSai 관리자 모드]

목적:
- 신고 처리
- 유해 콘텐츠 확인
- 개인정보 노출 시도 확인
- 계정 제재
- 서비스 품질 관리

볼 수 있어야 하는 것:
- 회원 목록
- 가입 상태
- 글/댓글 전체 목록
- 신고된 글/댓글/대화
- 차단/신고 기록
- 개인정보 차단 로그
- 관계 단계 상태
- 프로필 사진 공개 여부
- 계정 제재 이력

주의:
- 1:1 대화 전체 열람은 기본 상시 열람이 아니라
  신고/위반/운영 필요 사유가 있을 때만 열람하는 구조가 좋다.
- 관리자 열람 기록은 반드시 로그로 남긴다.
- 누가, 언제, 어떤 사용자 정보를 봤는지 저장한다.
- 프로필 사진 원본도 필요 시에만 확인 가능하게 한다.
- 개인정보처리방침에 운영자 검토 가능 범위를 명시한다.

필수 기능:
1. 대시보드
2. 회원 검색
3. 글/댓글 검색
4. 신고 목록
5. 대화 위반 로그
6. 계정 정지/해제
7. 글 숨김/삭제
8. 사용자 경고 발송
9. 관리자 열람 로그
10. 운영 메모

관리자 권한:
- owner: 전체 접근
- moderator: 신고/콘텐츠 처리
- support: 문의/계정 상태 확인
